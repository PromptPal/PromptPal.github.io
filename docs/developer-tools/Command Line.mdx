In order to improve the development experience. you need to install one more CLI: [https://github.com/PromptPal/cli](https://github.com/PromptPal/cli)

just download it from [release page](https://github.com/PromptPal/cli/releases) and move to `/usr/local/bin`. you can run it with `promptpal`

# Usage

## Init

```bash
promptpal init
```

this command will create a `promptpal.yaml` file in the current directory.

and you need to edit it to update endpoint and project Open Token.

once the configuration correct you can run `promptpal g` to generate type definitions.

## Generate

```bash
promptpal g
```

only we are only support JavaScript and Go. please check it in next page
