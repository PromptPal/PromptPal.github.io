## Github Action

```yaml
name: Testing

on:
  push:
    branches: [master]
  pull_request:
    branches: [master]

jobs:
  container-job:
    steps:
      - uses: PromptPal/github-action@v1.0.3
        env:
          PROMPTPAL_ENDPOINT: ${{ secrets.PROMPTPAL_ENDPOINT }}
          PROMPTPAL_API_TOKEN: ${{ secrets.PROMPTPAL_API_TOKEN }}
        with:
          token: ${{ secrets.GITHUB_TOKEN }}
          # generate types by PromptPal CLI
          command: pp g
```
